# Лабораторна робота №2

## Framework
Electron

## Опис проєкту

Кросплатформений настільний застосунок на Electron «Калькулятор кредиту та депозиту». Він дозволяє робити фінансові розрахунки і збирає результат у читабельний формат з можливістю збереження в текстовий файл.

Проект складається з:
- `src/main.js` — головний процес Electron, вікно, обробка подій та збереження файлу через діалог.
- `src/preload.js` — безпечний міст (contextBridge) для викликів з відрендереного UI до main process.
- `src/index.html` — інтерфейс користувача (форма для введення параметрів, кнопки, блок результату).
- `src/renderer.js` — логіка розрахунку (кредит/депозит), валідація, рендер результату, збереження.
- `src/style.css` — стилі для макета (картка, кнопки, повідомлення).

## Детальніше: що робить програма

1. При старті Electron відкриває вікно `index.html` розміром 720x680.
2. Користувач вибирає режим:
   - `credit` (кредит)
   - `deposit` (депозит)
3. Вводить параметри:
   - початкова сума (`principal`)
   - річна відсоткова ставка (`rate`)
   - термін у роках (`years`)
   - щомісячний платіж/внесок (`monthly`) для депозиту або кредиту
4. Натискає «Розрахувати».

### Кредитний розрахунок
- `monthlyRate = rate / 100 / 12`
- `months = years * 12`
- щомісячний платіж ануїтетним методом
- загальна сума виплат = `monthlyPayment * months`
- переплата = загальна сума виплат − principal

### Депозитний розрахунок
- просто копіюється `monthlyContribution` (може бути 0)
- нарощування (капіталізація) за формулою:
  - `finalAmount = principal * (1 + monthlyRate)^months + monthlyContribution * ((accumulation - 1)/monthlyRate)`
  - якщо ставка 0 → `finalAmount = principal + monthlyContribution * months`
- доход від відсотків = `finalAmount - (principal + monthlyContribution * months)`

5. Результат виводиться в `div#result` у форматі тексту (з валютним форматуванням Ukraine UAH) і зберігається в `lastResultText`.
6. Кнопка «Зберегти результати у файл» активується лише після успішного розрахунку.
7. При натисканні збереження викликається `window.electronAPI.saveResultFile(lastResultText)`, що відкриває діалог вибору шляху файлу.
8. Файл записується синхронним `fs.writeFileSync(filePath, resultText, 'utf8')`.

## Інструкція запуску

1. Встановити Node.js (рекомендовано 18+).
2. Перейти в директорію проекту:
   - `cd c:\Users\Admin\Desktop\RIK\Lab2`
3. Встановити залежності:
   - `npm install`
4. Запустити:
   - `npm start`

## Поради

- Увімкніть `mode=deposit`, щоб відразу вказувати суму щомісячно (можна 0).
- Якщо `rate=0`, розрахунок працює без формули штрафів і лише із сумою внесків.
