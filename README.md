# Лабораторна робота №2

## Framework
Electron

## Опис проєкту

У межах лабораторної роботи було розроблено кросплатформенний настільний застосунок із використанням фреймворку Electron.

Застосунок реалізує проведення опитування користувача з п’ятьма текстовими питаннями. Після завершення опитування результати зберігаються у текстовому файл формату `.txt` на диску.

## Як працює програма

1. При запуску відкривається вікно з першим питанням.
2. Користувач відповідає на питання у текстових полях.
3. Між питаннями можна переходити кнопками "Назад" і "Далі".
4. На кожному кроці відбувається перевірка введених даних (не порожні поля).
5. Після відповіді на останнє питання формується текстовий звіт.
6. Користувач може зберегти результат у файл `.txt` через діалог збереження.

## Функціональні можливості

- Відображення 5 питань
- Навігація між питаннями ("Назад" / "Далі")
- Перевірка введених даних
- Формування текстового звіту
- Збереження результатів у файл `.txt`

## Інструкція запуску

1. Встановити Node.js
2. Встановити залежності:
   - `npm install`
3. Запустити програму:
   - `npm start`

> Примітка: у цьому проєкті використовується стандартна команда `npm install` для встановлення залежностей, до яких належить Electron.
